# Brewery Finder

A specialized location-based search tool for the craft beer scene.

## Description

This application is a focused search engine designed to help users navigate the dense landscape of Chicago's craft breweries. By integrating with a live database, it allows users to filter by brewery type (micro, brewpub, etc.), see which breweries have food or don't, and view maps, providing instant access to addresses and contact information in a clean, mobile-responsive interface.

## Why I Built It

Living in a city with a world-class brewing scene, I wanted to build a tool that simplified the discovery process. Drawing on my background in Behavioral Psychology, I focused on reducing "search friction" by creating a UI that prioritizes essential information without unnecessary visual noise. This project serves as the technical foundation for a larger "Happy Hour" ecosystem, where I plan to integrate live event tracking (trivia, jazz jams, and bingo) to support both local bar and restaurant businesses and the music community, both of which I have been a part of in my life.

## Tech Stack

**Frontend:** React & Vite
**Backend:** Node.js & Express.js
**Data Source:** OpenBreweryDB API (Fetching live, categorized brewery data)
**Architecture:** Asynchronous API handling with custom filtering logic
**Design:** Custom CSS focused on brand identity and mobile-first responsiveness
